# Online Consultancy Service Application

A full-stack online homeopathy consultancy application developed as an **industry project with Maars Solutions**. This platform allows users to book consultations, access personalized treatment plans, communicate securely with practitioners, and conduct live video consultations. The application is designed for **scalability, security, and cross-platform compatibility**.
