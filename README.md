# Online Consultancy Service Application

A full-stack online homeopathy consultancy application developed as an **industry project with Maars Solutions**. This application allows users to book consultations, access personalized treatment plans, and communicate securely with practitioners. The platform is designed for scalability, security, and cross-platform compatibility.
